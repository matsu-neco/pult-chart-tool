# 🎻 プルト表作成ツール (Pult Chart Tool)

オーケストラの舞台配置（プルト表）を直感的に作成・管理できるWebツールです。
ハーモニーホールふくいなどの実際の舞台図を背景に、椅子や譜面台を配置できます。

## 🚀 公開URL
(ここにGitHub Pagesの設定後に表示されるURLを貼ってください)

## ✨ 主な機能
- **直感的な操作**: ドラッグ＆ドロップで楽器や椅子を配置。
- **複数選択・移動**: Shift + クリックで複数選択し、まとめて移動や削除が可能。
- **コピペ対応**: Ctrl+C / Ctrl+V でセクションを素早く複製。
- **保存・読込**: ブラウザの保存機能（LocalStorage）を利用し、作業内容を保存。
- **画像書き出し**: 作成した図面をPNG形式でダウンロード。
- **舞台図切り替え**: 特定のホールの図面を背景に設定可能。

## 🖱 使い方
- **楽器追加**: 画面上部のボタンをクリック。
- **名前入力**: アイコンをダブルクリックして奏者名やパート名を入力。
- **サイズ変更**: 右上のプルダウンから「大・中・小」を選択して追加。
- **背景変更**: ホールのリストから背景を選択。
